Catering Menu Manager

Update menu item names and prices, then publish or export BrightSign RSS feeds.

How to use this page

  1. Edit the item name or price.
  2. Use Copy Feed URL if BrightSign needs the RSS link.
  3. Click Publish Changes.

If publishing is not set up on this computer yet, use Download Update Package and give the ZIP file to the person who manages GitHub.

Menu Items

Each menu item has its own public RSS feed URL for BrightSign. Use your GitHub Pages URL, not a github.com repo link.
